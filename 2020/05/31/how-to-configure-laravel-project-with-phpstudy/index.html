    • how to configure laravel project with phpstudy

    前言

    开启本教程之前，你需要安装phpstudy

    并且已经存在laravel项目

    开启 openssl 配置

    在php目录下的php.ini里，将extension=php_openssl.dll前面的分号去掉即可，博主用的是phpstudy，如图



    小tips：Ctrl+F 快捷方式可实现 快速查找 功能

    顺便开启拓展配置php_mbstring和php_openSSl

    将项目移动到WWW下

    因为phpstudy默认文件入口是phpstudy/phptutorial/www

    配置域名

    在phpstudy界面中，我们点击 其他选项->站点域名管理

    网站目录指向laravel项目里的public目录（注意：Laravel 应用的所有请求入口都是 public/index.php 文件），
    这里网站域名随你填写，只是为了下一步添加进host里，方便访问

    接下来打开本地hosts文件，增加 【 127.0.0.1 域名 】，对域名进行映射

    phpstudy里面有快捷打开方式：点击 其他选项菜单-> 打开host 即可编写

    如果找不到，也可以本地搜索地址打开：C:\WINDOWS\system32\drivers\etc

    完成以后，在浏览器输入刚刚的域名，即可访问到我们的laravel项目了。
