    前言

    Laravel作为国际性的PHP开发框架，它具有的优点：

    • 实操性强
    • 灵活多变
    • 安全性能佳…

    安装laravel，你的电脑必须符合以下条件：

    • PHP> = 7.1.3
    • BCMath PHP扩展
    • Ctype PHP扩展
    • JSON PHP扩展
    • Mbstring PHP扩展
    • OpenSSL PHP扩展
    • PDO PHP扩展
    • Tokenizer PHP扩展
    • XML PHP扩展

    如果嫌配置麻烦，你可以利用已经集成好的工具来作为你的开发环境。如官方推荐的Laravel Homestead虚拟机，里面已经帮你配置好了上面所需，而且不喜欢还可以一键更新或删除。

    安装composer

    laravel是依靠composer管理依赖的，因此点击进入 composer 网站，点击下载

    Windows用户点击第一个就行了。下载完你会得到exe程序，每次点击它会自动帮你更新成最新的composer

    安装成功后，打开cmd输入composer，查看安装是否成功，成功界面如下。

    composer安装laravel

    有俩个办法，一是通过laravel安装器安装，安装速度较快；另一个是直接composer加载一个laravel项目，比较方便。
